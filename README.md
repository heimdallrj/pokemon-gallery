## Getting Started

### Setting up locally and installing dependencies

```bash
git clone git@github.com:thinkholic/pokemon-gallery.git
cd pokemon-gallery
npm install // or yarn
```

### Start application

```bash
npm start // yarn start
```

### Formatting the codebase

```
npm run format
```

### Tests

```bash
npm run test
```
